主题介绍

leaf是一个干净、具有现代感、完全响应式设计的wordpress主题。leaf 是专门为杂志、新闻网站和博客设计的。笔者经过两个月的使用再一次进行了修改和优化，感谢汉化作者Inhu。

因为是响应式的设计，所以Leaf适合所有平板、手机等移动设备阅读，并且极具简单的风格，一个简洁的主题可以很好的为读者提供纯净的阅读空间，足够干净的阅读空间和简练的排版是笔者最为看重的。

leaf汉化优化版

汉化优化部分

  1. 修改、优化关于字体多个样式
  2. 修改大部分样式以便适应中文。
  3. 修改程序对中文的截断。
  4. 首页、列表等页面的文章略缩图可以通过自定义栏目使用外链图片。
  5. 原版中，如果文章上了首页轮换，那么首页其他区域（非侧边栏）便不再出现。修改后，两个互为独立。
  6. 首页文章略缩图规定大小，如果图片大于则居中溢出隐藏。

首页的缩略图可以使用外链图片。外链缩略图使用方法：

添加自定义栏目leaf_image_medium和leaf_image_slider即可。

  • leaf_image_medium ： 中、小缩略图
  • leaf_image_slider ： 轮换大图

后台解释

  • Theme Layout 主题布局，其中分为boxed、wide。直接自己设置一下，好理解。
  • Show sidebar in single 这个是笔者加上去的，内容页是否显示侧边栏。
  • Theme Accent Color 主题的强调色。
  • Google Fonts 是否使用谷歌字体，建议勾选。中文不支持这个。
  • Slider Category 首页顶部轮播的分类。
  • Slider transition effects 轮播的切换方式。
  • Slider speed 轮播切换的时间。
  • Sidebar Column 侧边栏占用屏幕的宽度，总宽度为12格。
  • Home Page Categories 首页显示的分类文章。
  • Number of More Articles 更多文章的数量，更多文章与Home Page Categories相斥。
  • Custom CSS 自定义样式。
