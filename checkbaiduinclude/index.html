WordPress教程：在文章页面显示该文章是否被百度收录

WordPress迷 WordPress教程 01-24 687 24
本站使用阿里云主机，独立IP、不限流量、速度快、稳定的可靠云服务器，有利于SEO！戳我了解！

这里首先要感谢奶嘴，昨天在群里面提起了这个功能，一直等着他出教程，结果各种网络不给力，在今天他出教程的时候，wordpress迷也在百度找到了，将其移植到wordpress里面来了。效果如下图：

WordPress教程：在文章页面显示该文章是否被百度收录

看到大大的红字了吗？百度未收录，具体方法其实很简单，我是百度上找的，这里直接给出wordpress实现方法吧。将一下代码放到functions.php中：

  1. <?php   
  2. /*
  3. * 检测网页是否被百度收录，返回1则表示收录 返回0表示没有收录  
  4. * @ param string $url 待检测的网址  
  5. */  
  6. function checkBaiduInclude($url){   
  7.     $url='http://www.baidu.com/s?wd='.$url;   
  8.     $curl=curl_init();   
  9.     curl_setopt($curl,CURLOPT_URL,$url);   
  10.     curl_setopt($curl,CURLOPT_RETURNTRANSFER,1);   
  11.     $rs=curl_exec($curl);   
  12.     curl_close($curl);   
  13.     if(!strpos($rs,'抱歉，没有找到')){   
  14.         return 1;   
  15.     }else{   
  16.         return 0;   
  17.     }      
  18. }  

这里就是利用它去百度查询，如果有结果就返回1，无结果就返回0，这样我们在需要用到的地方来判断这个函数的返回值就OK了。在文章页面需要的地方放上如下代码：

  1. <?php    
  2.  if(checkBaiduInclude('get_permalink()') == 1)   
  3.  echo "百度已收录";   
  4.  else  
  5.  echo "百度未收录";    
  6.  ?>  

提交到百度收录的也很简单，如：

http://zhanzhang.baidu.com/sitesubmit/index?sitename=http://www.wpmee.com/maitang/

一看就明白了，将后面的网站换成你需要提交，也就是本文的链接就好了。

注意符号半角全角，这样点击一下就能提交到百度那边输入验证码就OK了。是不是很方便？当然本站使用了base64_encode加解密的跳转链接，有需要可以百度，实在不行，我可以放上来。

参考资料1：http://www.phpernote.com/php-function/842.html
参考资料2：http://v7v3.com/wpjiaocheng/201401566.html

PS：修复查询收录不准，将 “抱歉，没有找” 到修改为 “没有找到” 即可。

PS：如果使用了相对链接就应该写成这样：

  1. if(checkBaiduInclude("http://www.wpmee.com".get_permalink($post->ID)) == 1)
