WordPress教程：如何自定义WordPress欢迎面板？

WordPress迷 WordPress教程 02-19 427 9
本站使用阿里云主机，独立IP、不限流量、速度快、稳定的可靠云服务器，有利于SEO！戳我了解！

WordPress一直以来有一个功能基本是被大家所忽略的，只有每次在升级完新的wordpress版本之后才会自动显示出来，一般情况下大家是选择隐藏此功能。或许你已经知道是什么了。对，没错，就是wordpress的wellcome欢迎面板。在后台的仪表板就可以看到。

wellcome panle

其实我们可以将这个利用起来，如果你是主题制作者，可能会有什么东西想展示，比如广告？比如主题使用方法等等？那么如何通过代码的方式控制wordpress欢迎面板的显示内容呢？今天的wordpress教程就是如何自定义WordPress欢迎面板？

Let's do it!只需将下面的代码添加到你的主题的functions.php即可：

  1.   
  2. //自定义欢迎面板   
  3. function rc_my_welcome_panel() {   
  4.   
  5.     ?>   
  6. <script type="text/javascript">   
  7. /* Hide default welcome message */  
  8. jQuery(document).ready( function($)    
  9. {   
  10.     $('div.welcome-panel-content').hide();   
  11. });   
  12. </script>   
  13.   
  14.     <div class="custom-welcome-panel-content">   
  15.         <h3><?php _e( '欢迎面板自定义控制台消息标题' ); ?></h3>   
  16.         <p class="about-description"><?php _e( '您可以将您的自定义文本，给主题使用者的提示，把一个广告，或者你的联系信息等等展示在这里。' ); ?></p>   
  17.         <div class="welcome-panel-column-container">   
  18.             <div class="welcome-panel-column">   
  19.                 <h4><?php _e( "这里展示另外一个小标题" ); ?></h4>   
  20.                 <a class="button button-primary button-hero load-customize hide-if-no-customize" href="http://www.wpmee.com"><?php _e( '访问WordPress迷' ); ?></a>   
  21.                     <p class="hide-if-no-customize"><?php printf( __( '或者, <a href="%s">编辑你的设置</a>' ), admin_url( 'options-general.php' ) ); ?></p>   
  22.             </div>   
  23.         </div>   
  24.         <div class="">   
  25.         <h3><?php _e( '如果你需要更多的空间' ); ?></h3>   
  26.         <p class="about-description">创建一个新的段落！</p>   
  27.         <p>在这里写下你的自定义消息。</p>   
  28.         </div>   
  29.     </div>   
  30.   
  31. <?php   
  32. }   
  33.   
  34. add_action( 'welcome_panel', 'rc_my_welcome_panel' );  

是不是非常简单呢？如此好的一块wordpress广告位，必须利用起来啊。
