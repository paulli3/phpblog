wordpress主题：Theme Codename H

主题介绍

首先放个图是不是有点windows live的风格啊，Theme Codename H 是一款免费的 WordPress 博客主题，主题官网也在使用这样的主题，适合于科技博客，科技资讯报道等站点，感谢作者的无私奉献，一起来看看有哪些特性？

主题特性

增强的评论体验：支持 WordPress 原生评论嵌套、@ 回复评论、Gravatar 头像、Trackback 精简格式。了解更多 »

图片浏览特效：基于 ColorBox（轻量级 jQuery 插件）的图片特效，无需打开新页即可浏览原图，并支持键盘操作。关于 ColorBox »

内置页码导航支持：就像 WP-PageNavi 那样，内置的页码导航将自动为首页、存档页、或者搜索页加上页码，但无需安装任何插件。

现代浏览器：支持主流现代浏览器，包括 IE8、Firefox、Safari、Google Chrome、Opera 等。

IE8 网页快讯：订阅博客的新方式，单击之后，即可第一时间在 IE8 收藏栏知晓博客的更新。

移动版 Mobile：Theme Codename H 移动版是专为现代智能手持设备设计的主题模版。支持浏览最新文章和评论、评论 Gravatar 头像，移动版也继承了桌面版的文章样式。了解更多 »

主题安装

安装 Theme Codename H：

  • 第一步 下载 Theme Codename H
  • 第二步 将 h 文件夹上传至 /wp-content/themes/ 目录
  • 第三步 登录 WordPress 后台，在主题页面启用 Theme Codename H

安装 Theme Codename H 移动版：

  • 第一步 下载 Theme Codename H
  • 第二步 将 m 文件夹上传至 WordPress / 根目录
  • 第三步 修改 m-functions.php 文件 第 2 行，将 http://m.livesino.net 改为自己的移动版域名

安装 Windows Live Wave 4 for WordPress 表情：

  • 第一步 下载 Theme Codename H
  • 第二步 将 smilies 文件夹替换 /wp-includes/images/smilies 目录
