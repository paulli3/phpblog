实现教程

①、制作读者墙页面

首先复制你主题的page.php页面，改名为readerwall.php，然后在顶部 添加如下代码：

  1. <?php   
  2. /*
  3. Template Name: Reader wall  
  4. */  
  5. ?>  

继续搜索<?php the_content('');?>，在这句代码前面添加下面的代码：

  1. <?php   
  2.   
  3.     $query="SELECT COUNT(comment_ID) AS cnt, comment_author, comment_author_url, comment_author_email FROM (SELECT * FROM $wpdb->comments LEFT OUTER JOIN $wpdb->posts ON ($wpdb->posts.ID=$wpdb->comments.comment_post_ID) WHERE comment_date > date_sub( NOW(), INTERVAL 24 MONTH ) AND user_id='0' AND comment_author_email != '改成你的邮箱账号' AND post_password='' AND comment_approved='1' AND comment_type='') AS tempcmt GROUP BY comment_author_email ORDER BY cnt DESC LIMIT 39";//大家把管理员的邮箱改成你的,最后的这个39是选取多少个头像，大家可以按照自己的主题进行修改,来适合主题宽度   
  4.   
  5.     $wall = $wpdb->get_results($query);   
  6.   
  7.     $maxNum = $wall[0]->cnt;   
  8.   
  9.     foreach ($wall as $comment)   
  10.   
  11.     {   
  12.   
  13.         $width = round(40 / ($maxNum / $comment->cnt),2);//此处是对应的血条的宽度   
  14.   
  15.         if( $comment->comment_author_url )   
  16.   
  17.         $url = $comment->comment_author_url;   
  18.   
  19.         else $url="#";   
  20.   $avatar = get_avatar( $comment->comment_author_email, $size = '36', $default = get_bloginfo('wpurl').'/avatar/default.jpg' );   
  21.   
  22.         $tmp = "<li><a target=\"_blank\" href=\"".$comment->comment_author_url."\">".$avatar."<em>".$comment->comment_author."</em> <strong>+".$comment->cnt."</strong></br>".$comment->comment_author_url."</a></li>";   
  23.   
  24.         $output .= $tmp;   
  25.   
  26.      }   
  27.   
  28.     $output = "<ul class=\"readers-list\">".$output."</ul>";   
  29.   
  30.     echo $output ;   
  31.   
  32. ?>  

②、添加超酷CSS3代码美化读者墙页面，在你主题的style.css里添加如下css代码美化读者墙界面

  1. .readers-list{line-height:18px;text-align:left;overflow:hidden;_zoom:1}   
  2. .readers-list li{width:200px;float:left;*margin-right:-1px}   
  3. .readers-list a,.readers-list a:hover strong{background-color:#f2f2f2;background-image:-webkit-linear-gradient(#f8f8f8,#f2f2f2);background-image:-moz-linear-gradient(#f8f8f8,#f2f2f2);background-image:linear-gradient(#f8f8f8,#f2f2f2)}   
  4. .readers-list a{position:relative;display:block;height:36px;margin:4px;padding:4px 4px 4px 44px;color:#999;overflow:hidden;border:#ccc 1px solid;border-radius:2px;box-shadow:#eee 0 0 2px}   
  5. .readers-list img,.readers-list em,.readers-list strong{-webkit-transition:all .2s ease-out;-moz-transition:all .2s ease-out;transition:all .2s ease-out}   
  6. .readers-list img{width:36px;height:36px;float:left;margin:0 8px 0 -40px;border-radius:2px}   
  7. .readers-list em{color:#666;font-style:normal;margin-right:10px}   
  8. .readers-list strong{color:#ddd;width:40px;text-align:rightright;position:absolute;rightright:6px;top:4px;font:bold 14px/16px microsoft yahei}   
  9. .readers-list a:hover{border-color:#bbb;box-shadow:#ccc 0 0 2px;background-color:#fff;background-image:none}   
  10. .readers-list a:hover img{opacity:.6;margin-left:0}   
  11. .readers-list a:hover em{color:#EE8B17;font:bold 12px/36px microsoft yahei}   
  12. .readers-list a:hover strong{color:#EE8B17;rightright:150px;top:0;text-align:center;border-right:#ccc 1px solid;height:44px;line-height:40px}
