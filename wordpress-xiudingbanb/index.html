wordpress教程：去掉修订版本清除冗余数据

WordPress迷 WordPress教程 12-18 438 10
本站使用阿里云主机，独立IP、不限流量、速度快、稳定的可靠云服务器，有利于SEO！戳我了解！

修订版本这玩意看到特不舒服，个人感觉意义不大，反而造成数据库大量垃圾数据，百度找了下方法，特此分享出来，给正在折腾WP的同学们。直接进入主题！

首先打开functions.php 这个文件，不用说了吧，玩WP的都知道，加入：

remove_action ( 'pre_post_update', 'wp_save_post_revision' );

看字面意思就是移出这自动修订版本这个功能，接下来就是删除数据库中冗余的数据了，你可以先查一下到底有多少条，进入数据库高级管理-phpMyAdmin后台，在上面一排菜单中点击“SQL”，执行SQL语句：

SELECT * FROM wp_posts WHERE post_type = "revision"

点击执行就可以看到了，同样，删除语句：

DELETE FROM wp_posts WHERE post_type = "revision"

顿时感觉世界又干净了。
