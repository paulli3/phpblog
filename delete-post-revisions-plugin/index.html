WordPress插件： Delete Post Revisions By Tricks Panda删除文章修订版本

插件介绍

之前也介绍过移出修订版本这个功能，绝大多数的wordpress站长都禁用了，因为如果你未禁用此功能，那么每当您保存，预览一个页面或者文章，WordPress都将保存所有以前的版本修订，以帮助您恢复以前的数据。例如，如果你有1篇文章，它有10个修订版本那么就有10份未使用的数据存储在数据库中。所以清除不需要的数据，保持wordpress高速运行是非常重要的一件事情。那么今天就像大家介绍一款这样的wordpress插件，用来清理文章修订版本的垃圾数据。

就像面板显示这样，安装完毕插件，进入设置界面你只需点击一个按钮，它将自动清除所有的修订版本，如果你不要的时候也可以将他卸载。

怎么安装？

  1. 上传插件到你的wordpress-' wp-content/plugins '目录，或下载并通过你的管理面板自动安装。
  2. 在插件页面激活并启用插件，最后在设置界面操作就OK了。

您还可以通过在你的WordPress插件管理面板搜​​索'Delete Post Revisions By Tricks Panda“来安装此wordpress插件。

如何通过代码实现禁用wordpress修订版本的功能呢？

在wordpress根目录中找到wp-config.php并打开，添加如下代码：

  1. define('WP_POST_REVISIONS', false);  

当然，如果你的历史记录中已经有了修订版本，可以通过前面的插件删除，也可以通过操作mysql数据库，但是不建议直接操作数据库，这里提供一下方法。

  1. DELETE FROM wp_postmeta WHERE post_id IN (SELECT id FROM wp_posts WHERE post_type = 'revision');   
  2. DELETE FROM wp_term_relationships WHERE object_id IN (SELECT id FROM wp_posts WHERE post_type='revision');   
  3. DELETE FROM wp_posts WHERE post_type='revision';
